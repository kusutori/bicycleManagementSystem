# 单车管理系统
## 介绍
大一下学期C语言小作业，单车管理系统

做的还不是很完善，同时也是上传GitHub的第一次尝试

## 使用
源文件`bicycle.c`，点击即可下载，本地存储我简单的使用了txt来存储，放在`txt`文件夹里面，分别是`bicycle.txt`用来存储单车信息(目前已清空)，`member.txt`用来存储会员的相关信息
